# Gestionnaire de Tâches TODO

Ce script Bash est un gestionnaire de tâches simple qui permet à l'utilisateur de créer, mettre à jour, supprimer, et afficher des tâches stockées dans un fichier texte.

## Fonctionnalités

- **Créer une nouvelle tâche** : Ajoute une nouvelle tâche avec un titre, une description, un lieu, une date d'échéance et une heure d'échéance.
- **Mettre à jour une tâche existante** : Permet de modifier les détails d'une tâche existante en utilisant son ID unique.
- **Supprimer une tâche** : Supprime une tâche du fichier en utilisant son ID unique.
- **Afficher toutes les informations sur une tâche** : Affiche les détails d'une tâche spécifique en utilisant son ID unique.
- **Lister les tâches d'un jour donné** : Affiche toutes les tâches prévues pour une date spécifique.
- **Rechercher une tâche par le titre** : Recherche et affiche les tâches qui correspondent au titre donné.
- **Afficher l'aide** : Affiche les options d'utilisation du script.

## Prérequis

- Bash
- Un système d'exploitation compatible Unix

## Utilisation

Pour utiliser le script, vous pouvez invoquer les commandes suivantes dans votre terminal :

./todo_script.sh [option] [argument]
